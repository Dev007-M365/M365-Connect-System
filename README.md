\# M365-Connect-System



A modern PowerShell 7–based administration tool for Microsoft 365 tenants.  

Includes a WPF graphical interface, automatic module installation, multi‑tenant support, and a built‑in self‑update system powered by GitHub.



\---



\## 🚀 Features



\### ✔ Modern WPF GUI  

A clean, responsive interface built with Windows Presentation Foundation (WPF).



\### ✔ Multi‑Tenant Support  

Load multiple tenants from a simple `tenant.csv` file.



\### ✔ Multiple Connection Profiles  

Choose between:

\- \*\*Graph only\*\*

\- \*\*Teams only\*\*

\- \*\*Full Admin\*\* (Graph + Exchange Online + PnP + Teams)



\### ✔ Automatic Module Installation  

If required modules are missing, the launcher installs them automatically:

\- Microsoft.Graph  

\- ExchangeOnlineManagement  

\- PnP.PowerShell  

\- MicrosoftTeams  



\### ✔ Self‑Updating System  

The tool checks GitHub for updates on every launch and automatically:

\- Downloads new versions  

\- Replaces local files  

\- Restarts the launcher  



\### ✔ PowerShell 7 Required  

Ensures modern module compatibility and avoids legacy issues.
